Portfolio Projects API

This project serves as a backend API for showcasing my personal portfolio projects. Built with Node.js and Express, it delivers structured data about each project — including title, description, and associated image — through a RESTful endpoint. The API is publicly accessible and is consumed by a static front-end interface to dynamically render project cards.
